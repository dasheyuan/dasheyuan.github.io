C++测试框架Catch

Catch简介

Catch 一种是 C++ 测试框架。
主要特点：

  • 配置简单，下载catch.hpp，然后#include即可。
  • 无需额外库依赖。只要能编译 C++ 98 和包含 C++ 标准库。
  • 编写测试用例与编写函数一样。
  • Section中测试用例是独立运行的。
  • 兼容测试用例传统风格和BDD风格：Given-When-Then。
  • 条件比较是只用一个核心的警告宏。使用了标准的C/C++ 运算符重载功能。
