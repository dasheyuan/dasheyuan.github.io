小组任务，安排整理一下3GPP R12网络架构的内容。今天查找的内容简单小结一下。

网络架构参考模型

1、非漫游架构

非漫游架构有两种参考模型，这两种模型区别不大，主要是S-GW和P-GW是否是一个独立的网元。
图1 Non-roaming architecture for 3GPP accesse
图2 Non-roaming architecture for 3GPP accesses. Single gateway configuration option

2、漫游架构

漫游架构有三种参考模型，其中第1种是本地路由业务，P-GW位于本地网络侧；第2和第3种是非本地路由业务，P-GW位于访问网络侧。第2与第3种的区别是AF（Application Function）在哪一侧。 图3 Roaming architecture for 3GPP accesses. Home routed traffic 图4 Roaming architecture for local breakout, with home operator’s application functions only 图5 Roaming architecture for local breakout, with visited operator’s application functions only
